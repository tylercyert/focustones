## What is FocusTones.co  

FocusTones.co is a web application designed to generate customizable binaural tones that help you enter a state of deep focus, creativity, or relaxation. Built with modern web technologies, it delivers a smooth, distraction-free experience for professionals, students, and creators who want to improve their concentration and mental performance through sound.  

## How Binaural Beats Work  

Binaural beats are created by playing two slightly different frequencies in each ear, which the brain perceives as a single “beat” frequency. This phenomenon, called brainwave entrainment, gently guides brain activity into specific states:  

- **Delta (0.5–4 Hz)** – Deep sleep, recovery, physical restoration  
- **Theta (4–8 Hz)** – Meditation, creative thinking, relaxation  
- **Alpha (8–12 Hz)** – Calm focus, ideal for studying or light work  
- **Beta (12–30 Hz)** – Active thinking, problem-solving, alertness  
- **Gamma (30+ Hz)** – Peak concentration, memory, high-level processing  

## Key Features  

### Audio Engine  
- Real-time binaural beat generation with no noticeable delay  
- Smooth fade-in/out for a seamless listening experience  
- True stereo channel separation for authentic binaural effect  
- Adjustable volume control  

### Professional Controls  
- Select from five brainwave ranges  
- Set the carrier frequency (80Hz–1000Hz) to your preference  
- Fine-tune the beat frequency for each range  
- Automatic saving and restoring of user settings  

### Enhanced Audio Options  
- Optional white noise overlay with independent volume control  
- Low-pass filtering for a softer listening experience  
- Automatic synchronization of noise and binaural tones  

### Visualization  
- Interactive 3D sphere built with Three.js  
- Colors adjust dynamically based on the selected brainwave range  
- Responsive design that works across devices  
- One-click start/stop playback  

## Technology Stack  

- **Frontend:** React 19, TypeScript, Vite  
- **Styling:** Tailwind CSS 4, Material Design 3  
- **3D Graphics:** Three.js with GLSL shaders  
- **Audio:** Web Audio API  
- **State Management:** Zustand  
- **Deployment:** Docker + Nginx
